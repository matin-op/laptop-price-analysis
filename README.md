# 💻 Laptop Pricing & Review Insights Dashboard

## 📌 Project Overview
This project is an end-to-end data analysis pipeline where laptop data was scraped, cleaned, analyzed, and visualized to extract meaningful insights about pricing, ratings, and customer reviews.

---

## ⚙️ Tools & Technologies
- Python (requests, BeautifulSoup, pandas)
- SQL
- Power BI

---

## 🔄 Workflow
1. Scraped laptop data from an e-commerce test website using Python
2. Cleaned and structured the data using pandas
3. Stored and analyzed data using SQL queries
4. Built an interactive dashboard in Power BI

---

## 📊 Key Insights
- Most laptops fall in the budget category (~55%)
- Premium laptops have slightly higher reviews
- No strong correlation between rating and price
- Few laptops dominate the high-price segment

---

## 📁 Files Included
- `scraper.py` → Data scraping & cleaning script
- `analysis.sql` → SQL queries used for analysis
- `laptops.csv` → Dataset
- `laptop_dashboard.pbix` → Power BI dashboard file
